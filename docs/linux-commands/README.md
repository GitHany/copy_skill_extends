## 基础导航

### cd - 切换目录

**基础用法**:
```bash
cd %{目录路径}%
```

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 返回上一次目录 | `cd -` | 切换回之前的目录 |
| 返回用户主目录 | `cd ~` | 快速回到家目录 |
| 上级目录 | `cd ..` | 切换到父目录 |
| 根目录 | `cd /` | 切换到文件系统根 |

### pwd - 显示当前目录

**基础用法**:
```bash
pwd
```

### ls - 列出目录内容

**基础用法**:
```bash
ls -la
```

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 按时间排序 | `ls -lt` | 最新修改的文件在前 |
| 按大小排序 | `ls -lS` | 大文件在前 |
| 递归显示 | `ls -lR` | 显示所有子目录 |
| 人类可读大小 | `ls -lh` | 显示 K、M、G 单位 |
| 显示隐藏文件 | `ls -a` | 包括 . 开头的文件 |

### mkdir - 创建目录

**基础用法**:
```bash
mkdir -p %{文件夹路径}%
```

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 设置权限创建 | `mkdir -m 755 %{目录}%` | 创建时指定权限 |
| 批量创建目录 | `mkdir -p dir/{sub1,sub2,sub3}` | 一次性创建多个子目录 |
| 创建并显示 | `mkdir -v %{目录}%` | 显示创建信息 |

## 文件操作

### cp - 复制文件

**基础用法**:
```bash
cp -r %{源文件夹}% %{目标目录}%/
```

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 保留权限复制 | `cp -rp %{源}% %{目标}%` | 保留权限和时间戳 |
| 交互式复制 | `cp -i %{源}% %{目标}%` | 覆盖前提示 |
| 显示进度 | `rsync -ah --info=progress2 %{源}% %{目标}%` | 显示复制进度 |

### mv - 移动/重命名文件

**基础用法**:
```bash
mv %{源文件}% %{新文件夹}%/
```

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 重命名文件 | `mv %{旧名}% %{新名}%` | 改名 |
| 交互式移动 | `mv -i %{源}% %{目标}%` | 覆盖前提示 |
| 批量重命名 | `rename 's/old/new/' *.txt` | 使用正则批量改名 |

### rm - 删除文件

**基础用法**:
```bash
rm %{文件}%
```

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 递归删除目录 | `rm -r %{目录}%` | 删除目录及内容 |
| 强制删除 | `rm -rf %{目录}%` | 不提示直接删除 |
| 交互式删除 | `rm -i %{文件}%` | 删除前确认 |
| 删除空目录 | `rmdir %{目录}%` | 只能删除空目录 |

### find - 查找文件

**基础用法**:
```bash
find %{哪个目录}% -name "%{文件名(支持正则表达式)}%"
```

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 按类型查找 | `find %{目录}% -type f -name "*.py"` | 查找 Python 文件 |
| 按时间查找 | `find %{目录}% -mtime -7` | 最近 7 天修改的文件 |
| 按大小查找 | `find %{目录}% -size +100M` | 大于 100M 的文件 |
| 查找并删除 | `find %{目录}% -name "*.tmp" -delete` | 查找并删除 |
| 查找并执行 | `find %{目录}% -name "*.log" -exec gzip {} \;` | 查找并压缩 |

### grep - 搜索文件内容

**基础用法**:
```bash
grep "%{查找的内容}%" %{在哪一个文件查找}%
```

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 递归搜索 | `grep -r "%{内容}%" %{目录}%` | 搜索目录下所有文件 |
| 忽略大小写 | `grep -i "%{内容}%" %{文件}%` | 不区分大小写 |
| 显示行号 | `grep -n "%{内容}%" %{文件}%` | 显示匹配行号 |
| 反向搜索 | `grep -v "%{内容}%" %{文件}%` | 显示不匹配的行 |
| 正则搜索 | `grep -E "%{正则}%" %{文件}%` | 使用正则表达式 |
| 统计次数 | `grep -c "%{内容}%" %{文件}%` | 显示匹配行数 |
| 前后N行 | `grep -A 3 -B 2 "%{内容}%" %{文件}%` | 显示上下文 |
| 只显示文件名 | `grep -l "%{内容}%" %{目录}/*.py` | 只列文件名 |

## 权限管理

### chmod - 修改权限

**基础用法**:
```bash
chmod %{权限}% %{文件}%
```

**权限说明**:
- `r` = 4 (读)
- `w` = 2 (写)
- `x` = 1 (执行)

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 设置 755 | `chmod 755 %{文件}%` | rwxr-xr-x |
| 设置 644 | `chmod 644 %{文件}%` | rw-r--r-- |
| 递归修改 | `chmod -R 755 %{目录}%` | 目录下所有文件 |
| 添加执行权限 | `chmod +x %{文件}%` | 添加执行权限 |
| 移除写权限 | `chmod -w %{文件}%` | 移除写权限 |

### chown - 修改所有者

**基础用法**:
```bash
chown %{用户}:%{组}% %{文件}%
```

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 修改所有者 | `chown %{用户}% %{文件}%` | 只改所有者 |
| 递归修改 | `chown -R %{用户}:%{组}% %{目录}%` | 递归修改 |
| 修改组 | `chgrp %{组}% %{文件}%` | 只修改组 |

## 归档压缩

### tar - 打包压缩

**基础用法**:
```bash
tar -czvf %{压缩包名称(.tar.gz)}%.tar.gz %{目录位置}%
```

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 解压 | `tar -xzvf %{压缩包}%.tar.gz` | 解压 tar.gz |
| 查看内容 | `tar -tzvf %{压缩包}%.tar.gz` | 列出内容 |
| 排除文件 | `tar -czvf %{名称}%.tar.gz --exclude='*.log' %{目录}%` | 排除日志 |
| 解压到指定目录 | `tar -xzvf %{名称}%.tar.gz -C %{目标目录}%` | 指定解压路径 |



## 💡 实用场景示例

### 场景 1: 日志分析

```bash
# 查找最近的错误日志
tail -f /var/log/app.log | grep -i "error"

# 统计错误出现次数
grep -c "ERROR" /var/log/app.log

# 查找包含特定内容的日志文件
grep -r "NullPointerException" /var/log/

# 查看大日志文件
less +F /var/log/app.log
```

### 场景 2: 磁盘清理

```bash
# 找出占用空间最大的目录
du -ah / | sort -rh | head -n 20

# 删除 7 天前的日志文件
find /var/log -name "*.log" -mtime +7 -delete

# 清理 apt 缓存
apt-get clean

# 查找并删除临时文件
find /tmp -type f -mtime +1 -delete
```

### 场景 3: 进程管理

```bash
# 查找占用 CPU 最高的进程
ps aux --sort=-%cpu | head -n 10

# 查找占用内存最高的进程
ps aux --sort=-%mem | head -n 10

# 终止某个应用的所有进程
pkill -f %{应用名}%

# 后台运行任务并保存日志
nohup %{命令}% > output.log 2>&1 &
```

### 场景 4: 网络排查

```bash
# 测试网络连通性
ping -c 4 google.com

# 查看端口占用
netstat -tulpn | grep %{端口}%
ss -tulpn | grep %{端口}%

# 查看 DNS 解析
nslookup %{域名}%
dig %{域名}%

# 追踪路由
traceroute %{主机}%
```
