# Redis 列表操作

Redis 列表操作常用命令参考。

## LPUSH 左侧插入

**命令**: `LPUSH %{键名}% %{值}%`

**参数说明**:
| 参数 | 类型 | 必填 | 说明 | 示例 |
|------|------|------|------|------|
| 键名 | string | 是 | 列表键名【常用】 | mylist |
| 值 | string | 是 | 要插入的值 | world |

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 左插入元素 | `redis-cli LPUSH mylist "world"` | 在列表头部插入值【常用】 |
| 查看列表 | `redis-cli LRANGE mylist 0 -1` | 查看列表所有元素 |

## LINDEX 按索引获取

**命令**: `LINDEX %{键名}% %{索引}%`

**参数说明**:
| 参数 | 类型 | 必填 | 说明 | 示例 |
|------|------|------|------|------|
| 键名 | string | 是 | 列表键名 | mylist |
| 索引 | integer | 是 | 元素索引（0为第一个，-1为最后一个） | 0 |

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 获取首元素 | `redis-cli LINDEX mylist 0` | 索引=0，获取第一个元素 |
| 获取尾元素 | `redis-cli LINDEX mylist -1` | 索引=-1，获取最后一个元素 |

## LTRIM 修剪列表

**命令**: `LTRIM %{键名}% %{开始}% %{结束}%`

**参数说明**:
| 参数 | 类型 | 必填 | 说明 | 示例 |
|------|------|------|------|------|
| 键名 | string | 是 | 列表键名 | mylist |
| 开始 | integer | 是 | 起始索引 | 0 |
| 结束 | integer | 是 | 结束索引（-1表示到末尾） | -1 |

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 保留前N个元素 | `redis-cli LTRIM mylist 0 9` | 保留索引0-9的元素 |
| 保留指定范围 | `redis-cli LTRIM mylist 0 -1` | 保留所有元素（等效于查看） |

---

## LREM 移除元素

**参数说明**:
| 参数 | 类型 | 必填 | 说明 | 示例 |
|------|------|------|------|------|
| 键名 | string | 是 | 列表键名 | mylist |

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 移除元素 | `redis-cli LREM mylist 1 "hello"` | 从列表中移除1个hello值 |

## LINSERT 插入元素

**命令**: `LINSERT %{键名}% BEFORE %{基准值}% %{新值}%`

**参数说明**:
| 参数 | 类型 | 必填 | 说明 | 示例 |
|------|------|------|------|------|
| 键名 | string | 是 | 列表键名 | mylist |
| 基准值 | string | 是 | 参照值（插入到此值之前或之后） | pivot |
| 新值 | string | 是 | 要插入的新值 | newitem |

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 在基准值之前插入 | `redis-cli LINSERT mylist BEFORE pivot newitem` | 在pivot之前插入newitem |
| 在基准值之后插入 | `redis-cli LINSERT mylist AFTER pivot newitem` | 在pivot之后插入newitem |
