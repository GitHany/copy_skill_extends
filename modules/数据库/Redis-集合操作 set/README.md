# Redis 集合操作

Redis 集合操作常用命令参考。

## SADD 添加集合成员

**命令**: `SADD %{键名}% %{成员}%`

**参数说明**:
| 参数 | 类型 | 必填 | 说明 | 示例 |
|------|------|------|------|------|
| 键名 | string | 是 | 集合键名【常用】 | myset |
| 成员 | string | 是 | 要添加的成员 | alice |

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 添加单个成员 | `redis-cli SADD myset "alice"` | 键名=myset, 成员=alice【常用】 |
| 添加多个成员 | `redis-cli SADD myset "alice" "bob" "charlie"` | 一次添加多个成员 |

|------|------|------|------|------|

## SSCAN 迭代集合

**命令**: `SSCAN %{键名}% %{游标}%`

**参数说明**:
| 参数 | 类型 | 必填 | 说明 | 示例 |
|------|------|------|------|------|
| 键名 | string | 是 | 集合键名 | myset |
| 游标 | integer | 是 | 游标位置（0表示开始新一轮迭代） | 0 |

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 迭代所有成员 | `redis-cli SSCAN myset 0` | 游标=0，开始迭代 |
| 指定匹配模式 | `redis-cli SSCAN myset 0 MATCH user:*` | 只返回匹配user:*的成员 |
| 指定返回数量 | `redis-cli SSCAN myset 0 COUNT 100` | 每次最多返回100个成员 |
