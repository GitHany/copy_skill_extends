## 基础导航

### cd - 切换目录

**基础用法**:
```bash
cd %{目录路径}%
```

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 返回上一次目录 | `cd -` | 切换回之前的目录 |
| 返回用户主目录 | `cd ~` | 快速回到家目录 |
| 上级目录 | `cd ..` | 切换到父目录 |
| 根目录 | `cd /` | 切换到文件系统根 |
| 指定目录 | `cd /var/log` | 目录路径，示例: /var/log |
| 上上级目录 | `cd ../..` | 切换到上两级目录 |
| 绝对路径 | `cd /home/user/projects` | 使用绝对路径切换 |
| 相对路径 | `cd ../sibling` | 使用相对路径切换 |

### find - 查找文件

**基础用法**:
```bash
find %{哪个目录}% -name "%{文件名(支持正则表达式)}%"
```

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 按类型查找 | `find %{目录}% -type f -name "*.py"` | -type f: 普通文件；-name: 文件名模式 【常用】 |
| 按时间查找 | `find %{目录}% -mtime -7` | -mtime -7: 7天内修改的文件 【常用】 |
| 按大小查找 | `find %{目录}% -size +100M` | -size +100M: 大于100M的文件 【常用】 |
| 查找并删除 | `find %{目录}% -name "*.tmp" -delete` | -delete: 删除找到的文件 |
| 查找并执行 | `find %{目录}% -name "*.log" -exec gzip {} \;` | -exec: 对每个文件执行命令 |
| 查找空目录 | `find %{目录}% -type d -empty` | -type d: 目录类型 |
| 查找符号链接 | `find %{目录}% -type l` | -type l: 符号链接 |
| 查找权限 | `find %{目录}% -type f -perm 644` | -perm 644: 权限为644的文件 |
| 排除目录 | `find %{目录}% -not -path "*/node_modules/*"` | -not -path: 排除匹配路径 |

### grep - 搜索文件内容

**基础用法**:
```bash
grep "%{查找的内容}%" %{在哪一个文件查找}%
```

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 递归搜索 | `grep -r "%{内容}%" %{目录}%` | -r: 递归搜索目录；示例: /var/log 【常用】 |
| 忽略大小写 | `grep -i "%{内容}%" %{文件}%` | -i: 不区分大小写 |
| 显示行号 | `grep -n "%{内容}%" %{文件}%` | -n: 显示匹配行号 【常用】 |
| 反向搜索 | `grep -v "%{内容}%" %{文件}%` | -v: 显示不匹配的行 |
| 正则搜索 | `grep -E "%{正则}%" %{文件}%` | -E: 使用正则表达式 |
| 统计次数 | `grep -c "%{内容}%" %{文件}%` | -c: 统计匹配行数 |
| 前后N行 | `grep -A 3 -B 2 "%{内容}%" %{文件}%` | -A/-B: 匹配后/前N行 |
| 只显示文件名 | `grep -l "%{内容}%" %{目录}/*.py` | -l: 只列文件名 |
| 统计匹配行数 | `grep -o "%{内容}%" %{文件}% | wc -l` | -o: 精确统计匹配次数 |
| 高亮显示 | `grep --color "%{内容}%" %{文件}%` | --color: 高亮匹配文本 |

## 权限管理

### chmod - 修改权限

**基础用法**:
```bash
chmod %{权限}% %{文件}%
```

**权限说明**:
- `r` = 4 (读)
- `w` = 2 (写)
- `x` = 1 (执行)

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 设置 755 | `chmod 755 %{文件}%` | rwxr-xr-x |
| 设置 644 | `chmod 644 %{文件}%` | rw-r--r-- |
| 递归修改 | `chmod -R 755 %{目录}%` | 目录下所有文件 |
| 添加执行权限 | `chmod +x %{文件}%` | 添加执行权限 |
| 移除写权限 | `chmod -w %{文件}%` | 移除写权限 |
| 所有者可读写 | `chmod u+rw %{文件}%` | 给所有者添加读写 |
| 组可读执行 | `chmod g+rx %{文件}%` | 给组添加读执行 |
| 其他人可读 | `chmod o=r %{文件}%` | 设置其他人只读 |
| 复制权限 | `chmod --reference=%{参考文件}% %{文件}%` | 复制其他文件权限 |

### chown - 修改所有者

**基础用法**:
```bash
chown %{用户}:%{组}% %{文件}%
```

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 修改所有者 | `chown %{用户}% %{文件}%` | 只改所有者 |
| 递归修改 | `chown -R %{用户}:%{组}% %{目录}%` | 递归修改 |
| 修改组 | `chgrp %{组}% %{文件}%` | 只修改组 |
| 仅修改组 | `chown :%{组}% %{文件}%` | 只修改所属组 |
| 递归保留 | `chown -R --from=:oldgroup %{用户}:%{组}% %{目录}%` | 保留特定组 |

## 归档压缩

### tar - 打包压缩

**基础用法**:
```bash
tar -czvf %{压缩包名称(.tar.gz)}%.tar.gz %{目录位置}%
```

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 解压 | `tar -xzvf %{压缩包}%.tar.gz` | -x: 解压；-z: gzip；-v: 详细；-f: 文件 【常用】 |
| 查看内容 | `tar -tzvf %{压缩包}%.tar.gz` | -t: 列出内容 【常用】 |
| 排除文件 | `tar -czvf %{名称}%.tar.gz --exclude='*.log' %{目录}%` | --exclude: 排除匹配文件 |
| 解压到指定目录 | `tar -xzvf %{名称}%.tar.gz -C %{目标目录}%` | -C: 指定解压路径 |
| 仅打包 | `tar -cvf %{名称}%.tar %{目录}%` | -c: 创建归档（无压缩） |
| 追加文件 | `tar -rvf %{名称}%.tar %{新增文件}%` | -r: 追加文件 |
| 保留时间戳 | `tar -p %{名称}%.tar.gz %{目录}%` | -p: 保留文件时间 |
| 多目录打包 | `tar -czvf backup.tar.gz /home /etc /var` | 打包多个目录 |
| 显示进度 | `tar -czvf - %{目录}% | pv | gzip > backup.tar.gz` | pv: 显示压缩进度 |
| 排除多个模式 | `tar -czvf archive.tar.gz --exclude='*.log' --exclude='temp/*' %{目录}%` | 排除多个模式 |

### zip/unzip - 压缩解压

**基础用法**:
```bash
zip -r %{压缩包名称}%.zip %{文件名称}%
unzip %{压缩包名称}%.zip -d %{目标目录}%
```

|------|------|------|
| 压缩文件 | `zip archive.zip file1.txt file2.txt` | 压缩多个文件 |
| 压缩目录 | `zip -r archive.zip folder/` | 递归压缩目录 |
| 设置压缩级别 | `zip -9 archive.zip %{文件}%` | 最大压缩比 |
| 排除文件 | `zip -r archive.zip folder/ -x "*.log"` | 排除特定文件 |
| 加密压缩 | `zip -e archive.zip %{文件}%` | 需要输入密码 |
| 查看内容 | `unzip -l archive.zip` | 列出压缩包内容 |
| 测试压缩 | `unzip -t archive.zip` | 测试压缩包完整性 |
| 覆盖已有 | `unzip -o archive.zip -d %{目录}%` | 覆盖已存在的文件 |

## 进程管理

### ps - 查看进程

**基础用法**:
```bash
ps aux
```

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 查看指定用户 | `ps -u %{用户名}%` | -u: 显示指定用户的进程 |
| 查看指定进程 | `ps -ef | grep %{进程名}%` | -ef: 完整格式；grep: 过滤 【常用】 |
| 树状显示 | `ps auxf` | auxf: 进程树结构显示 【常用】 |
| 按 CPU 排序 | `ps aux --sort=-%cpu | head -n 10` | --sort=-%cpu: CPU降序 |
| 按内存排序 | `ps aux --sort=-%mem | head -n 10` | --sort=-%mem: 内存降序 |
| 显示线程 | `ps -eLf | grep %{PID}%` | -eLf: 显示线程 |

## 💡 实用场景示例

### 场景 1: 日志分析

```bash
# 查找最近的错误日志
tail -f /var/log/app.log | grep -i "error"

# 统计错误出现次数
grep -c "ERROR" /var/log/app.log

# 查找包含特定内容的日志文件
grep -r "NullPointerException" /var/log/

# 查看大日志文件
less +F /var/log/app.log
```

### 场景 2: 磁盘清理

```bash
# 找出占用空间最大的目录
du -ah / | sort -rh | head -n 20

# 删除 7 天前的日志文件
find /var/log -name "*.log" -mtime +7 -delete

# 清理 apt 缓存
apt-get clean

# 查找并删除临时文件
find /tmp -type f -mtime +1 -delete
```

### 场景 3: 进程管理

```bash
# 查找占用 CPU 最高的进程
ps aux --sort=-%cpu | head -n 10

# 查找占用内存最高的进程
ps aux --sort=-%mem | head -n 10

# 终止某个应用的所有进程
pkill -f %{应用名}%

# 后台运行任务并保存日志
nohup %{命令}% > output.log 2>&1 &
```

### 场景 4: 网络排查

```bash
# 测试网络连通性
ping -c 4 google.com

# 查看端口占用
netstat -tulpn | grep %{端口}%
ss -tulpn | grep %{端口}%

# 查看 DNS 解析
nslookup %{域名}%
dig %{域名}%

# 追踪路由
traceroute %{主机}%
```
