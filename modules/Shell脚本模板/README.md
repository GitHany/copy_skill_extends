## find 文件查找

### find - 查找文件

**基础用法**:
```bash
find %{路径}% %{选项}% -name "%{文件名}%"
```

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 按名称查找 | `find %{目录}% -name "*.log"` | -name: 文件名模式 【常用】 |
| 按类型查找 | `find %{目录}% -type f -name "*.txt"` | -type f: 普通文件 |
| 按大小查找 | `find %{目录}% -size +100M` | -size +100M: 大于100M的文件 【常用】 |
| 按时间查找 | `find %{目录}% -mtime -7` | -mtime -7: 7天内修改的文件 【常用】 |
| 查找并执行 | `find %{目录}% -name "*.log" -exec gzip {} \;` | -exec: 对每个文件执行命令 |
| 排除目录 | `find %{目录}% -not -path "*/node_modules/*"` | -not -path: 排除匹配路径 |

## grep 文本搜索

### grep - 搜索文件内容

**基础用法**:
```bash
grep "%{模式}%" %{文件路径}%
```

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 递归搜索 | `grep -r "%{内容}%" %{目录}%` | -r: 递归搜索目录 【常用】 |
| 显示行号 | `grep -n "%{内容}%" %{文件}%` | -n: 显示匹配行号 【常用】 |
| 忽略大小写 | `grep -i "%{内容}%" %{文件}%` | -i: 不区分大小写 |
| 仅显示文件名 | `grep -l "%{内容}%" %{目录}/*.log` | -l: 只列文件名 |
| 正则搜索 | `grep -E "%{正则}%" %{文件}%` | -E: 使用正则表达式 【常用】 |
| 统计匹配行数 | `grep -c "%{内容}%" %{文件}%` | -c: 统计匹配行数 |

## awk 数据处理

### awk - 数据处理

**基础用法**:
```bash
awk '%{模式}%' %{文件路径}%
```

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 打印指定列 | `awk '{print $1,$3}' %{文件}%` | $1: 第1列；$3: 第3列 【常用】 |
| 按分隔符提取 | `awk -F',' '{print $2}' %{文件}%` | -F: 指定分隔符 【常用】 |
| 条件筛选 | `awk '/pattern/ {print $0}' %{文件}%` | 只输出匹配的行 |
| 数值计算 | `awk '{sum+=$1} END {print sum}' %{文件}%` | 统计求和 |
| 格式化输出 | `awk 'printf "%s: %.2f\n", $1, $2' %{文件}%` | printf 格式化 【常用】 |
| 多条件判断 | `awk '$3 > 100 {print $1}' %{文件}%` | 按条件过滤 |

## sed 流编辑

### sed - 文本替换

**基础用法**:
```bash
sed 's/%{原字符串}%/%{新字符串}%/%{标志}%' %{文件路径}%
```

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 全局替换 | `sed 's/old/new/g' %{文件}%` | g: 替换所有匹配 【常用】 |
| 原地替换 | `sed -i 's/old/new/g' %{文件}%` | -i: 直接修改文件 【常用】 |
| 删除行 | `sed '2d' %{文件}%` | 删除第2行 |
| 插入文本 | `sed '1i\new line' %{文件}%` | 在第1行前插入 |
| 追加文本 | `sed '$a\new line' %{文件}%` | 在末尾追加 |
| 正则替换 | `sed -E 's/[0-9]+/NUM/g' %{文件}%` | -E: 正则表达式 【常用】 |

## xargs 参数传递

### xargs - 命令行参数转换

**基础用法**:
```bash
xargs %{选项}% %{命令}%
```

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 批量删除 | `find . -name "*.tmp" | xargs rm` | 将文件名作为参数传递 【常用】 |
| 逐个处理 | `echo "a b c" | xargs -n 1 echo` | -n 1: 每次传一个参数 【常用】 |
| 配合 grep | `grep -l "pattern" *.log | xargs rm` | 删除匹配的文件 |
| 询问确认 | `cat list.txt | xargs -p rm` | -p: 执行前确认 |
| 指定分隔符 | `echo "a,b,c" | xargs -d',' echo` | -d: 指定输入分隔符 |

## tar 压缩解压

### tar - 打包压缩

**基础用法**:
```bash
tar %{动作}% %{压缩包}%.tar.gz %{路径}%
```

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 解压 | `tar -xzvf %{压缩包}%.tar.gz` | -x: 解压；-z: gzip；-v: 详细；-f: 文件 【常用】 |
| 查看内容 | `tar -tzvf %{压缩包}%.tar.gz` | -t: 列出内容 【常用】 |
| 排除文件 | `tar -czvf %{名称}%.tar.gz --exclude='*.log' %{目录}%` | --exclude: 排除匹配文件 |
| 解压到目录 | `tar -xzvf %{名称}%.tar.gz -C %{目标目录}%` | -C: 指定解压路径 【常用】 |
| 追加文件 | `tar -rvf %{名称}%.tar %{新增文件}%` | -r: 追加文件 |
| 仅打包 | `tar -cvf %{名称}%.tar %{目录}%` | -c: 创建归档（无压缩） |

## zip/unzip 打包

### zip/unzip - 压缩解压

**基础用法**:
```bash
zip %{压缩包名称}%.zip %{文件名称}%
unzip %{压缩包名称}%.zip -d %{目标目录}%
```

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 压缩目录 | `zip -r archive.zip folder/` | -r: 递归压缩 【常用】 |
| 解压 | `unzip archive.zip` | 解压到当前目录 【常用】 |
| 解压到目录 | `unzip archive.zip -d /target/dir` | -d: 指定目标目录 |
| 查看内容 | `unzip -l archive.zip` | 列出压缩包内容 |
| 排除文件 | `zip -r archive.zip folder/ -x "*.log"` | -x: 排除特定文件 |
| 最大压缩比 | `zip -9 archive.zip file.txt` | -9: 最大压缩 |

## cut 文本切割

### cut - 提取字段

**基础用法**:
```bash
cut %{选项}% %{文件路径}%
```

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 按列提取 | `cut -d',' -f1,3 %{文件}%` | -d: 分隔符；-f: 字段号 【常用】 |
| 提取单列 | `cut -d':' -f1 /etc/passwd` | 提取第一列 |
| 按字节提取 | `cut -b 1-10 %{文件}%` | -b: 按字节位置 【常用】 |
| 按字符提取 | `cut -c 1-5 %{文件}%` | -c: 按字符位置 |
| 排除字段 | `cut -d':' --complement -f1 /etc/passwd` | 显示除第一列外的所有 |
| 输出分隔符 | `cut -d',' -f1,3 -o' | ' %{文件}%` | -o: 自定义输出分隔符 |

## head/tail 文件头尾

### head - 查看文件开头

**基础用法**:
```bash
head -n %{行数}% %{文件}%
```

### tail - 查看文件末尾

**基础用法**:
```bash
tail -f %{文件}%
```

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 查看前N行 | `head -n 20 %{文件}%` | 查看前20行 【常用】 |
| 实时跟踪 | `tail -f %{文件}%` | 持续显示新增内容 【常用】 |
| 查看最后N行 | `tail -n 100 %{文件}%` | 查看最后100行 【常用】 |
| 从第N行开始 | `tail -n +100 %{文件}%` | 从第100行开始 |

## 💡 实用场景示例

### 场景 1: 日志分析

```bash
# 查找错误日志
grep -i "error" /var/log/syslog

# 统计错误出现次数
grep -c "ERROR" /var/log/app.log

# 提取关键日志字段
awk '/ERROR/ {print $1, $2, $NF}' /var/log/app.log

# 查看最近100行
tail -n 100 /var/log/app.log
```

### 场景 2: 数据处理

```bash
# 提取CSV第2列并排序
cut -d',' -f2 data.csv | sort

# 去重统计
sort data.txt | uniq -c | sort -rn

# 计算总和
awk -F',' '{sum+=$3} END {print sum}' data.csv

# 格式化输出
awk '{printf "%-10s %5.2f\n", $1, $2}' data.txt
```

### 场景 3: 文件批量操作

```bash
# 批量压缩日志
find /var/log -name "*.log" -mtime +7 | xargs gzip

# 备份并排除临时文件
tar -czvf backup.tar.gz --exclude='*.tmp' /home/user

# 查找并替换文本
find . -name "*.txt" | xargs sed -i 's/old/new/g'
```

### 场景 4: 文本统计

```bash
# 统计文件行数
wc -l *.log

# 统计各类型文件数量
find . -type f | wc -l

# 最长行的长度
wc -L /var/log/syslog

# 统计目录大小
du -sh /var/log/*
```
