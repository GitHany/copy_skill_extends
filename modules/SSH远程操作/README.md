## ssh 远程连接

通过SSH协议连接到远程主机。

**基础用法**:
```bash
ssh %{用户}%@%{主机}%
```

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 指定端口连接 | `ssh -p %{端口}% %{用户}%@%{主机}%` | 端口: SSH端口号 (例: 2222)；用户: 用户名 (例: root)【常用】；主机: 主机地址 (例: 192.168.1.100)【常用】 |
| 使用密钥文件连接 | `ssh -i %{密钥文件}% %{用户}%@%{主机}%` | 密钥文件: 私钥文件路径 (例: ~/.ssh/id_rsa)；用户: 用户名 (例: root)【常用】；主机: 主机地址 (例: 192.168.1.100)【常用】 |
| 指定密钥文件并指定端口 | `ssh -p %{端口}% -i %{密钥文件}% %{用户}%@%{主机}%` | 端口: SSH端口号 (例: 2222)；密钥文件: 私钥文件路径 (例: ~/.ssh/id_rsa) |
| 跳板机连接 | `ssh -J %{跳板用户}%@%{跳板主机}% %{用户}%@%{主机}%` | 跳板用户: 跳板机用户名 (例: admin)；跳板主机: 跳板机地址 (例: bastion.example.com)；用户: 目标用户名 (例: root)【常用】；主机: 目标主机地址 (例: 192.168.1.100)【常用】 |
| 后台保持连接 | `ssh -o ServerAliveInterval=60 %{用户}%@%{主机}%` | 用户: 用户名 (例: root)【常用】；主机: 主机地址 (例: 192.168.1.100)【常用】 |

## scp 远程复制

通过SSH协议安全地复制文件到远程主机或从远程主机复制。

**基础用法**:
```bash
scp %{源文件}% %{用户}%@%{主机}%:%{目标路径}%
```

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 复制目录（递归） | `scp -r %{源目录}% %{用户}%@%{主机}%:%{目标路径}%` | 源目录: 源目录路径 (例: ./mydir)；用户: 用户名 (例: root)【常用】；主机: 主机地址 (例: 192.168.1.100)【常用】；目标路径: 目标路径 (例: /home/user/)【常用】 |
| 指定端口复制 | `scp -P %{端口}% %{源文件}% %{用户}%@%{主机}%:%{目标路径}%` | 端口: SSH端口号 (例: 2222) |
| 使用密钥文件复制 | `scp -i %{密钥文件}% %{源文件}% %{用户}%@%{主机}%:%{目标路径}%` | 密钥文件: 私钥文件路径 (例: ~/.ssh/id_rsa) |
| 从远程主机下载到本地 | `scp %{用户}%@%{主机}%:%{远程文件}% %{本地路径}%` | 用户: 用户名 (例: root)【常用】；主机: 主机地址 (例: 192.168.1.100)【常用】；远程文件: 远程文件路径 (例: /home/user/file.txt)【常用】；本地路径: 本地目标路径 (例: ./downloads/)【常用】 |
| 保留文件属性 | `scp -p %{源文件}% %{用户}%@%{主机}%:%{目标路径}%` | 源文件: 源文件路径 (例: file.txt)【常用】；用户: 用户名 (例: root)【常用】；主机: 主机地址 (例: 192.168.1.100)【常用】；目标路径: 目标路径 (例: /home/user/)【常用】 |
| 限速复制 | `scp -l %{限速值}% %{源文件}% %{用户}%@%{主机}%:%{目标路径}%` | 限速值: 限速值(Kbit/s) (例: 1024) |

## sftp 远程文件传输

通过SSH协议进行安全的文件传输。

**基础用法**:
```bash
sftp %{用户}%@%{主机}%
```

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 指定端口连接 | `sftp -P %{端口}% %{用户}%@%{主机}%` | 端口: SSH端口号 (例: 2222)；用户: 用户名 (例: root)【常用】；主机: 主机地址 (例: 192.168.1.100)【常用】 |
| 使用密钥文件连接 | `sftp -i %{密钥文件}% %{用户}%@%{主机}%` | 密钥文件: 私钥文件路径 (例: ~/.ssh/id_rsa)；用户: 用户名 (例: root)【常用】；主机: 主机地址 (例: 192.168.1.100)【常用】 |
| 批处理模式执行命令 | `sftp -b %{命令文件}% %{用户}%@%{主机}%` | 命令文件: 批处理命令文件路径 (例: ./sftp_commands.txt)；用户: 用户名 (例: root)【常用】；主机: 主机地址 (例: 192.168.1.100)【常用】 |

## ssh-keygen 生成SSH密钥对

生成、管理SSH密钥对。

**基础用法**:
```bash
ssh-keygen -t rsa -b 4096
```

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 生成RSA密钥对 | `ssh-keygen -t rsa -b %{位数}% -C "%{注释}%"` | 位数: 密钥位数 (例: 4096)【常用】；注释: 密钥注释(邮箱) (例: user@example.com) |
| 生成Ed25519密钥对 | `ssh-keygen -t ed25519 -C "%{注释}%"` | 注释: 密钥注释(邮箱) (例: user@example.com) |
| 指定密钥保存路径 | `ssh-keygen -t %{类型}% -f %{密钥路径}%` | 类型: 密钥类型 (例: rsa)【常用】；密钥路径: 密钥保存路径 (例: ~/.ssh/mykey) |
| 添加密码保护密钥 | `ssh-keygen -p -f %{密钥文件}%` | 密钥文件: 私钥文件路径 (例: ~/.ssh/id_rsa)【常用】 |
| 查看公钥 | `ssh-keygen -y -f %{私钥文件}%` | 私钥文件: 私钥文件路径 (例: ~/.ssh/id_rsa) |
| 显示指纹 | `ssh-keygen -lf %{密钥文件}%` | 密钥文件: 密钥文件路径 (例: ~/.ssh/id_rsa.pub) |
| 清除passphrase | `ssh-keygen -p -N "" -f %{密钥文件}%` | 密钥文件: 私钥文件路径 (例: ~/.ssh/id_rsa) |

## ssh-copy-id 复制公钥到远程主机

将本地公钥复制到远程主机的authorized_keys实现无密码登录。

**基础用法**:
```bash
ssh-copy-id %{用户}%@%{主机}%
```

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 指定端口复制公钥 | `ssh-copy-id -p %{端口}% %{用户}%@%{主机}%` | 端口: SSH端口号 (例: 2222)；用户: 用户名 (例: root)【常用】；主机: 主机地址 (例: 192.168.1.100)【常用】 |
| 指定密钥文件复制公钥 | `ssh-copy-id -i %{公钥文件}% %{用户}%@%{主机}%` | 公钥文件: 公钥文件路径 (例: ~/.ssh/id_rsa.pub)；用户: 用户名 (例: root)【常用】；主机: 主机地址 (例: 192.168.1.100)【常用】 |
| 复制到指定SSH路径 | `ssh-copy-id -o "IdentityFile=%{密钥文件}%" %{用户}%@%{主机}%` | 密钥文件: 私钥文件路径 (例: ~/.ssh/id_rsa)；用户: 用户名 (例: root)【常用】；主机: 主机地址 (例: 192.168.1.100)【常用】 |

## 实用场景示例

### 场景1: 配置SSH免密码登录

```bash
# 1. 生成密钥对
ssh-keygen -t ed25519 -C "user@example.com"

# 2. 复制公钥到远程主机
ssh-copy-id root@192.168.1.100

# 3. 测试连接（无需输入密码）
ssh root@192.168.1.100
```

### 场景2: 传输文件到远程服务器

```bash
# 上传单个文件
scp file.txt root@192.168.1.100:/home/user/

# 上传整个目录
scp -r ./mydir root@192.168.1.100:/home/user/

# 从远程下载到本地
scp root@192.168.1.100:/home/user/file.txt ./
```

### 场景3: 通过非标准端口连接

```bash
# SSH连接（非标准端口）
ssh -p 2222 root@192.168.1.100

# SCP上传（端口2222）
scp -P 2222 file.txt root@192.168.1.100:/home/user/

# SFTP连接（端口2222）
sftp -P 2222 root@192.168.1.100
```

### 场景4: 使用跳板机访问内网主机

```bash
# 通过跳板机连接内网服务器
ssh -J admin@bastion.example.com root@192.168.1.100

# 通过跳板机复制文件
scp -o "ProxyJump=admin@bastion.example.com" file.txt root@192.168.1.100:/home/user/
```

### 场景5: 使用特定密钥连接

```bash
# 指定私钥文件连接
ssh -i ~/.ssh/mykey root@192.168.1.100

# 指定私钥文件上传
scp -i ~/.ssh/mykey file.txt root@192.168.1.100:/home/user/
```
