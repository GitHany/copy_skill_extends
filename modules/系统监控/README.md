## top/htop 进程监控

### top - 实时进程监控

**基础用法**:
```bash
top %{选项}%
```

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 交互式进程监控 | `htop` | htop 提供更友好的交互界面 【常用】 |
| 指定用户进程 | `top -u %{用户名}%` | -u: 只显示指定用户的进程 【常用】 |
| 指定PID监控 | `top -p %{PID}%` | -p: 监控特定进程 【常用】 |
| 按CPU排序 | `top -o %CPU` | -o: 按指定列排序 |
| 按内存排序 | `top -o %MEM` | 按内存使用率排序 |
| 设置刷新间隔 | `top -d %{秒}%` | -d: 设置刷新间隔秒数 |
| 只看运行中 | `top -R` | -R: 不显示空闲进程 |
| 显示线程 | `top -H` | 显示所有线程 |
| 显示完整命令 | `top -c` | -c: 显示完整命令行 |

## vmstat 虚拟内存统计

### vmstat - 系统内存、进程、CPU统计

**基础用法**:
```bash
vmstat %{间隔}% %{次数}%
```

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 显示内存详情 | `vmstat -a` | -a: 显示活跃/非活跃内存 【常用】 |
| 显示磁盘统计 | `vmstat -d` | -d: 显示磁盘读写统计 【常用】 |
| 显示扩展统计 | `vmstat -x` | -x: 显示详细扩展信息 【常用】 |
| 显示 slab 缓存 | `vmstat -m --sort=-%s | head -n 20` | 查看内核 slab 缓存 【常用】 |
| 显示 MB 单位 | `vmstat -m` | 以 MB 为单位显示 |
| 显示详细统计 | `vmstat -s` | 显示事件计数器和内存统计 |

## iostat I/O统计

### iostat - 磁盘I/O和CPU统计

**基础用法**:
```bash
iostat %{选项}% %{间隔}% %{次数}%
```

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 显示CPU统计 | `iostat -c` | -c: 只显示 CPU 统计 【常用】 |
| 显示磁盘统计 | `iostat -d` | -d: 只显示磁盘统计 【常用】 |
| 显示扩展统计 | `iostat -x` | -x: 显示扩展设备统计 【常用】 |
| 人类可读格式 | `iostat -h` | -h: 人类可读格式 |
| 指定设备 | `iostat -p %{设备}%` | -p: 只显示指定设备 【常用】 |
| 持续监控 | `iostat -xz 2 10` | 每2秒采样，共10次 【常用】 |
| 所有设备 | `iostat -p ALL` | 显示所有分区和设备 |

## free 内存使用

### free - 内存使用情况

**基础用法**:
```bash
free %{选项}%
```

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 人类可读格式 | `free -h` | 显示 K、M、G 单位 【常用】 |
| 持续监控 | `free -h -s %{秒}%` | 每隔指定秒刷新 【常用】 |
| 显示总计 | `free -ht` | 显示 swap 总计行 【常用】 |
| 实时监控 | `watch -n 1 free -h` | 每秒更新显示 |
| 以MB为单位 | `free -m` | 以 MB 为单位 |
| 低内存告警 | `free -th | awk '/Mem/{print $3/$2*100}'` | 计算内存使用率 |

## df/du 磁盘使用

### df - 磁盘空间使用

**基础用法**:
```bash
df %{选项}% %{路径}%
```

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 人类可读格式 | `df -h` | 显示 K、M、G 单位 【常用】 |
| 显示所有文件系统 | `df -a` | 包括虚拟文件系统 【常用】 |
| 显示文件系统类型 | `df -T -t ext4` | 显示 ext4/xfs 等类型 【常用】 |
| 显示 inode 使用 | `df -i` | inode 使用情况 【常用】 |
| 查看指定目录 | `df -h %{路径}%` | 显示指定挂载点 |

### du - 目录大小

**基础用法**:
```bash
du %{选项}% %{路径}%
```

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 目录总大小 | `du -sh %{路径}%` | 显示目录总大小 【常用】 |
| 子目录大小排序 | `du -sh %{路径}%/*/ \| sort -hr` | 按大小排序显示子目录 【常用】 |
| 最大文件列表 | `du -ah %{路径}% \| sort -rh \| head -n 10` | 找出最大的文件 【常用】 |
| 显示层级 | `du -h --max-depth=1` | 显示一层深度 【常用】 |
| 排除特定目录 | `du -sh --exclude='node_modules' %{路径}%` | 排除指定目录 |
| 所有文件大小 | `du -ah %{路径}%` | 包括普通文件 |

## watch 周期性执行

### watch - 周期性执行命令

**基础用法**:
```bash
watch %{选项}% %{命令}%
```

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 每2秒执行 | `watch -n 2 %{命令}%` | 每隔2秒执行一次 【常用】 |
| 高亮差异 | `watch -d %{命令}%` | 高亮显示变化的部分 【常用】 |
| 持续监控内存 | `watch -n 1 free -h` | 每秒监控内存使用 【常用】 |
| 持续监控进程 | `watch -n 2 'ps aux \| head -1'` | 监控进程变化 |
| 持续监控磁盘 | `watch -n 5 df -h` | 每5秒监控磁盘空间 |
| 持续监控网络 | `watch -n 2 'ss -tulpn'` | 监控网络连接变化 |

## lsof 查看打开的文件

### lsof - 列出打开的文件

**基础用法**:
```bash
lsof %{选项}%
```

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 查看所有打开文件 | `lsof` | 显示所有打开的文件 【常用】 |
| 查看指定用户 | `lsof -u %{用户名}%` | 显示指定用户打开的文件 【常用】 |
| 查看指定进程 | `lsof -p %{PID}%` | 显示指定进程打开的文件 【常用】 |
| 查看网络连接 | `lsof -i` | 显示所有网络连接 【常用】 |
| 查看TCP连接 | `lsof -i TCP` | 显示 TCP 连接 【常用】 |
| 查看端口占用 | `lsof -i :%{端口}%` | 显示指定端口的占用 【常用】 |
| 递归查看目录 | `lsof +D %{目录}%` | 查看目录下所有打开的文件 【常用】 |
| 查看文件被谁用 | `lsof %{文件路径}%` | 查看哪个进程在使用文件 |
| 查看DNS查询 | `lsof -i :53` | 查看 DNS 服务端口 |

## 实用场景示例

### 场景 1: 排查内存问题

```bash
# 查看内存使用情况
free -h

# 查看内存和交换空间
free -ht

# 持续监控内存
watch -n 1 free -h

# 查看详细内存统计
vmstat -s
```

### 场景 2: 排查磁盘问题

```bash
# 查看磁盘空间
df -h

# 查看 inode 使用
df -i

# 找出最大目录
du -sh /* | sort -hr | head -n 10

# 持续监控磁盘
watch -n 5 df -h
```

### 场景 3: 排查CPU和进程问题

```bash
# 查看进程
htop

# 按CPU排序查看进程
top -o %CPU

# 按内存排序查看进程
top -o %MEM

# 查看特定用户进程
top -u www-data
```

### 场景 4: 排查I/O问题

```bash
# 查看磁盘I/O统计
iostat -x 2 5

# 查看每个磁盘
iostat -d -k 1 5

# 查看CPU统计
iostat -c
```

### 场景 5: 排查端口和文件问题

```bash
# 查看端口占用
lsof -i :80

# 查看网络连接
lsof -i TCP

# 查看某个用户的所有网络连接
lsof -i -u www-data

# 查看某个进程打开的文件
lsof -p 1234
```
