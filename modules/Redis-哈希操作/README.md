# Redis 哈希操作

Redis 哈希表操作常用命令参考。

## HSET 设置哈希字段

**命令**: `HSET %{键名}% %{字段}% %{值}%`

**参数说明**:
| 参数 | 类型 | 必填 | 说明 | 示例 |
|------|------|------|------|------|
| 键名 | string | 是 | 哈希键名【常用】 | user:1001 |
| 字段 | string | 是 | 哈希字段名 | name |
| 值 | string | 是 | 字段值 | 张三 |

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 设置哈希字段 | `redis-cli HSET user:1001 name "张三"` | 键名=user:1001, 字段=name, 值=张三【常用】 |
| 批量设置 | `redis-cli HMSET user:1001 name 张三 age 25 city 北京` | 键名=user:1001, 交替传入字段值对 |

## HMSET 批量设置哈希字段

**命令**: `HMSET %{键名}% %{field_value_pairs}%`

**参数说明**:
| 参数 | 类型 | 必填 | 说明 | 示例 |
|------|------|------|------|------|
| 键名 | string | 是 | 哈希键名 | user:1001 |
| field_value_pairs | string | 是 | 字段值对列表（交替排列） | name 张三 age 25 city 北京 |

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 批量设置多个字段 | `redis-cli HMSET user:1001 name 张三 age 25 city 北京` | 一次设置多个字段值 |

## HMGET 批量获取哈希字段

**命令**: `HMGET %{键名}% %{字段1}% %{字段2}%`

**参数说明**:
| 参数 | 类型 | 必填 | 说明 | 示例 |
|------|------|------|------|------|
| 键名 | string | 是 | 哈希键名 | user:1001 |
| 字段1 | string | 是 | 第一个字段 | name |
| 字段2 | string | 是 | 第二个字段 | age |

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 批量获取字段 | `redis-cli HMGET user:1001 name age` | 键名=user:1001, 获取name和age字段 |

## HDEL 删除哈希字段

**命令**: `HDEL %{键名}% %{字段1}% %{字段2}%`

**参数说明**:
| 参数 | 类型 | 必填 | 说明 | 示例 |
|------|------|------|------|------|
| 键名 | string | 是 | 哈希键名 | user:1001 |
| 字段1 | string | 是 | 第一个字段 | email |
| 字段2 | string | 否 | 第二个字段 | phone |

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 删除单个字段 | `redis-cli HDEL user:1001 email` | 键名=user:1001, 删除email字段 |
| 删除多个字段 | `redis-cli HDEL user:1001 email phone` | 批量删除多个字段 |

## HINCRBY 递增哈希字段值

**命令**: `HINCRBY %{键名}% %{字段}% %{增量}%`

**参数说明**:
| 参数 | 类型 | 必填 | 说明 | 示例 |
|------|------|------|------|------|
| 键名 | string | 是 | 哈希键名 | user:1001 |
| 字段 | string | 是 | 字段名 | age |
| 增量 | integer | 是 | 增量值（负数为递减） | 1 |

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 字段值加指定数 | `redis-cli HINCRBY user:1001 age 1` | age字段值+1，负数则递减 |
