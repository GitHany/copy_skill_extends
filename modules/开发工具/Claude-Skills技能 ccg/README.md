## ccg 代码生成

Claude-Codex-Gemini 三模型协作

**基础用法**:
```
/ccg %{任务描述}%
```

**扩展示例**:

| 场景 | 命令 | 说明 |
|------|------|------|
| 代码生成 | `/ccg 生成 REST API 代码` | 任务描述：生成 REST API 代码 |
| 快速模式 | `/ccg 生成用户管理 REST API --fast` | 任务：生成用户管理 REST API |
